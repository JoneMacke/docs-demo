# 📖 Easydict

::: info Software Info
**Official Website:** [Easydict （Click to go）](https://github.com/tisfeng/Easydict)  
**SupportPlatform:** MacOS
:::

## 📝 Configuration Steps

1. **Service** → **Openai Translation** → **CustomServiceprovider**
2. **API Endpoint:** `https://ai.katioai.com/v1/chat/completions`
3. **API Key:** Enter your token/API key
4. **model:** manuallyAdd Model

::: warning Notes
- ✅ API TypeSelect Openai Translation
- ✅ fill in the complete address
- ❌ Do not omit the model prefix/suffix
:::
