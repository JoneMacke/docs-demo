# 🖱️ Cursor

::: info Software Info
**Official Website:** [Cursor (Click to go)](https://cursor.com/)  
**SupportPlatform:** Windows / macOS / Linux
:::

## 📝 Configuration Steps

1. **Settings** → **Models** → **manuallyAddConfiguration**
2. **API mode:** OpenAI
3. **API address:** `https://ai.katioai.com/v1`
5. **API Key:** Enter your token/API key
6. **model (Model):** manuallyAdd Model

::: warning Notes
- ✅ Make sure API Fill in the addressCorrect，include `/v1`
- ❌ Do not omit the model prefix/suffix
:::
