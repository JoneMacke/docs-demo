# ⚙️ n8n

::: info Software Info
**Official Website:** [n8n (Click to go)](https://n8n.io/)  
**SupportPlatform:** Windows / macOS / web
:::

## 📝 Configuration Steps

1. **Settings** → **Addmodule** → **manuallyAddConfiguration**
2. **API mode:** text
3. **API Host:** `https://ai.katioai.com/v1`
4. **module:** `Message a model`
5. **API Key:** Enter your token/API key
6. **model (Model):** click to fetch models

::: warning Notes
- ✅ Make sure API Fill in the addressCorrect，include `/v1`
- ❌ NoteAddmodule
:::
