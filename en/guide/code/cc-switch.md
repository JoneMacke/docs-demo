# 🔀 CC Switch

::: info Software Info
**Official Website:** [CC Switch (Click to go)](https://github.com/farion1231/cc-switch/releases)  
**SupportPlatform:** Windows / macOS / Linux
:::

## 📝 Configuration Steps

1. **Select ModelService** → **SelectAdd** → **Addprovided byprovider**
2. **API address:** `https://ai.katioai.com`
3. **API Key:** Enter your token/API key
4. **main model:** Select ModelAdd
5. **API format:** anthropic messages（native）


::: warning Notes
- ✅ Make sure API Fill in the addressCorrect，not include `/v1`
- ❌ Do not omit the model prefix/suffix
:::
