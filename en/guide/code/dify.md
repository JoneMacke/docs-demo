# Dify

::: info Software Info
**Official Website:** [Dify (Click to go)](https://cloud.dify.ai/apps)  
**SupportPlatform:** Windows / macOS / web
:::

## 📝 Configuration Steps

1. **Settings** → **AddPlugin** → **manuallyAddConfiguration**
2. **API mode:** openai compatibility mode
3. **API Host:** `https://ai.katioai.com/v1`
4. **Plugin:** `openai-api-compatible`
5. **API Key:** Enter your token/API key
6. **model (Model):** click to fetch models

::: warning Notes
- ✅ Make sure API Fill in the addressCorrect，include `/v1`
- ❌ NoteAddPlugin
:::
