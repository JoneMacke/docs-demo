## 📝 Configuration Steps

1. **Settings** → **Serviceprovided byprovider** → **AddCustomServiceprovider**
2. **API Server:** `https://ai.katioai.com`
3. **API Key:** Enter your token/API key
4. **model(Models):** obtainModel List/manuallyAdd

::: warning Notes
- ✅ Make sure API Fill in the addressCorrect，not include `/v1`
- ❌ Do not omit the model prefix/suffix
:::
