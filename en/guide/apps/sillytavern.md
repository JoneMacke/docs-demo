## 📝 Configuration Steps

1. **API connect** → **chat completion** → **Custom（compatibleOpenai）**
2. **Customendpoint:** `https://ai.katioai.com/v1`
3. **API Key:** Enter your token/API key
4. **model:** obtainmodel

::: warning Notes
- ✅ API TypeSelect Chat Completion
- ✅ sourceSelect Custom (OpenAI-compatible)
- ❌ Do not omit the model prefix/suffix
:::
