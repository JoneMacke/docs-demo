## 📝 Configuration Steps

1. **Settings** → **modelprovider** → **manuallyAddConfiguration**
2. **API mode:** OpenAI API compatible
3. **API Host:** `https://ai.katioai.com/v1`
4. **API Path:** `/chat/completions`
5. **API Key:** Enter your token/API key
6. **model (Model):** click to fetch models/manuallyAdd

::: warning Notes
- ✅ check all model capabilities,Do not omit the model prefix/suffix
- ❌ do not check unless necessary"improve network compatibility"
:::
