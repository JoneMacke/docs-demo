## 📝 Configuration Steps

1. **Settings** → **provider** → **AddConfiguration**
2. **API mode:** OpenAI
3. **API Host:** `https://ai.katioai.com/v1`
4. **API Path:** `/chat/completions`
5. **API Key:** Enter your token/API key
6. **model (Model):** click to fetch models/manuallyAdd

::: warning Notes
- ✅ Make sure API Fill in the addressCorrect，include `/v1`
- ❌ Do not omit the model prefix/suffix
:::
