## 📝 Configuration Steps

1. **More** → **API link** → **AddCustomService**
2. **API address:** `https://ai.katioai.com/v1`
3. **API Key:** Enter your token/API key
4. **model:** obtainModel List

::: warning Notes
- ✅ Make sure API Fill in the addressCorrect，include `/v1`
- ❌ Do not omit the model prefix/suffix
:::
