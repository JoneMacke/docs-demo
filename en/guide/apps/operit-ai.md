## 📝 Configuration Steps

1. **Settings** → **model andparameterConfiguration** → **AddOtherprovider**
2. **API address:** `https://ai.katioai.com/v1/chat/completions`
3. **API Key:** Enter your token/API key
4. **model:** obtainModel List

::: warning Notes
- ✅ Make sure API Fill in the addressCorrect
- ❌ Do not omit the model prefix/suffix
:::
