# 🐾 Copaw

::: info 软件信息
**官方网站：** [Copaw (点击跳转)](https://copaw.agentscope.io/downloads)  
**支持平台：** Windows / Mac OS / Linux
:::

## 📝 配置步骤

1. **更多** → **API 链接** → **添加自定义服务**
2. **API 地址：** `https://ai.katioai.com/v1`
3. **API Key：** 填入您的令牌/密钥
4. **模型：** 获取模型列表

::: warning 注意事项
- ✅ 确保 API 地址填写正确，包含 `/v1`
- ❌ 不要遗漏模型的前后缀
:::
