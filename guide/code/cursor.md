# 🖱️ Cursor

::: info 软件信息
**官方网站：** [Cursor (点击跳转)](https://cursor.com/)  
**支持平台：** Windows / macOS / Linux
:::

## 📝 配置步骤

1. **设置** → **Models** → **手动添加配置**
2. **API 模式：** OpenAI
3. **API 地址：** `https://api.katioai.com/v1`
5. **API Key：** 填入您的令牌/密钥
6. **模型 (Model)：** 手动添加模型

::: warning 注意事项
- ✅ 确保 API 地址填写正确，包含 `/v1`
- ❌ 不要遗漏模型的前后缀
:::
