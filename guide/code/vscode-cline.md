# 💻 VS Code (Cline)

Cline 是一款强大的 VS Code AI 编程插件，支持自定义 API。

## 功能特点

- 代码生成与编辑
- 文件操作与管理
- 终端命令执行
- 多模型支持

## 配置枫叶AI

1. 在 VS Code 中安装 Cline 插件
2. 打开 Cline 设置
3. 选择 API Provider → OpenAI Compatible
4. 填入：
   - Base URL: `https://api.example.com/v1`
   - API Key: 你的枫叶AI密钥
   - Model: `maple-ai-v1`
