# ⚙️ n8n

::: info 软件信息
**官方网站：** [n8n (点击跳转)](https://n8n.io/)  
**支持平台：** Windows / macOS / web
:::

## 📝 配置步骤

1. **设置** → **添加模块** → **手动添加配置**
2. **API 模式：** text
3. **API 主机：** `https://api.katioai.com/v1`
4. **模块：** `Message a model`
5. **API Key：** 填入您的令牌/密钥
6. **模型 (Model)：** 点获取拉取模型

::: warning 注意事项
- ✅ 确保 API 地址填写正确，包含 `/v1`
- ❌ 注意添加模块
:::
