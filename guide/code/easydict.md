# 📖 Easydict

::: info 软件信息
**官方网站：** [Easydict （点击跳转）](https://github.com/tisfeng/Easydict)  
**支持平台：** MacOS
:::

## 📝 配置步骤

1. **服务** → **Openai 翻译** → **自定义服务商**
2. **接口地址：** `https://api.katioai.com/v1/chat/completions`
3. **API Key：** 填入您的令牌/密钥
4. **模型：** 手动添加模型

::: warning 注意事项
- ✅ API 类型选择 Openai 翻译
- ✅ 地址填写完整
- ❌ 不要遗漏模型的前后缀
:::
