# 🔀 CC Switch

::: info 软件信息
**官方网站：** [CC Switch (点击跳转)](https://github.com/farion1231/cc-switch/releases)  
**支持平台：** Windows / macOS / Linux
:::

## 📝 配置步骤

1. **选择模型服务** → **选择添加** → **添加提供商**
2. **API 地址：** `https://api.katioai.com`
3. **API Key：** 填入您的令牌/密钥
4. **主模型：** 选择模型添加
5. **API 格式：** anthropic messages（原生）


::: warning 注意事项
- ✅ 确保 API 地址填写正确，不包含 `/v1`
- ❌ 不要遗漏模型的前后缀
:::
