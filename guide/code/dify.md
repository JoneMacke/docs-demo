# Dify

::: info 软件信息
**官方网站：** [Dify (点击跳转)](https://cloud.dify.ai/apps)  
**支持平台：** Windows / macOS / web
:::

## 📝 配置步骤

1. **设置** → **添加插件** → **手动添加配置**
2. **API 模式：** openai 兼容模式
3. **API 主机：** `https://api.katioai.com/v1`
4. **插件：** `openai-api-compatible`
5. **API Key：** 填入您的令牌/密钥
6. **模型 (Model)：** 点获取拉取模型

::: warning 注意事项
- ✅ 确保 API 地址填写正确，包含 `/v1`
- ❌ 注意添加插件
:::
