## 📝 获取步骤

1. **登录** → [枫叶AI控制台](https://ai.katioai.com/console)
2. **进入** → 「API 管理」页面
3. **点击** → 「创建新密钥」
4. **复制** → 妥善保存你的 API Key

::: warning 注意事项
- ✅ 请妥善保存 API Key
- ❌ 请勿在前端代码中暴露你的 API Key，建议通过后端代理转发请求
:::
