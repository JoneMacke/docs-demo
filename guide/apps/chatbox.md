## 📝 配置步骤

1. **设置** → **模型提供方** → **手动添加配置**
2. **API 模式：** OpenAI API 兼容
3. **API 主机：** `https://ai.katioai.com/v1`
4. **API 路径：** `/chat/completions`
5. **API Key：** 填入您的令牌/密钥
6. **模型 (Model)：** 点获取拉取模型/手动添加

::: warning 注意事项
- ✅ 勾选所有模型能力,不要遗漏模型的前后缀
- ❌ 非必要不要勾选"改善网络兼容性"
:::
