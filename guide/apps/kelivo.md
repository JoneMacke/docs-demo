## 📝 配置步骤

1. **设置** → **供应商** → **添加配置**
2. **API 模式：** OpenAI
3. **API 主机：** `https://api.katioai.com/v1`
4. **API 路径：** `/chat/completions`
5. **API Key：** 填入您的令牌/密钥
6. **模型 (Model)：** 点获取拉取模型/手动添加

::: warning 注意事项
- ✅ 确保 API 地址填写正确，包含 `/v1`
- ❌ 不要遗漏模型的前后缀
:::
