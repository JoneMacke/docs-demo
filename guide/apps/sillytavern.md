## 📝 配置步骤

1. **API 连接** → **聊天补全** → **自定义（兼容Openai）**
2. **自定义端点：** `https://ai.katioai.com/v1`
3. **API Key：** 填入您的令牌/密钥
4. **模型：** 获取模型

::: warning 注意事项
- ✅ API 类型选择 Chat Completion
- ✅ 来源选择 Custom (OpenAI-compatible)
- ❌ 不要遗漏模型的前后缀
:::
