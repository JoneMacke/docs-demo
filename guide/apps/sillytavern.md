## 📝 配置步骤

1. **API 连接** → **Chat Completion** → **Custom (OpenAI-compatible)**
2. **Custom Endpoint：** `https://api.katioai.com/v1`
3. **API Key：** 填入您的令牌/密钥
4. **模型：** 手动输入模型名称（包含前后缀）

::: warning 注意事项
- ✅ API 类型选择 Chat Completion
- ✅ 源选择 Custom (OpenAI-compatible)
- ❌ 不要遗漏模型的前后缀
:::
