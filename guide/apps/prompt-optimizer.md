# 提示词优化器

::: info 软件信息
**官方网站：** [提示词优化器](https://prompt.always200.com/#/basic/system)  
**支持平台：** web
:::

## 📝 配置步骤

1. **模型管理** → **添加** → **添加自定义服务**
2. **API 地址：** `https://api.katioai.com/v1`
3. **API Key：** 填入您的令牌/密钥
4. **模型：** 获取模型列表

::: warning 注意事项
- ✅ 确保 API 地址填写正确，包含 `/v1`
- ❌ 不要遗漏模型的前后缀
:::
