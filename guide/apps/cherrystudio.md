## 📝 配置步骤

1. **打开设置** → **选择添加** → **添加提供商**
2. **API 地址：** `https://api.katioai.com`
3. **API Key：** 填入您的令牌/密钥
4. **模型：** 选择获取模型列表

::: warning 注意事项
- ✅ 确保 API 地址填写正确，不包含 `/v1`
- ❌ 不要遗漏模型的前后缀
:::
