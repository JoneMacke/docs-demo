## 📝 配置步骤

1. **设置** → **模型服务商** → **添加自定义提供商**
2. **API 地址：** `https://api.katioai.com/v1`
3. **API Key：** 填入您的令牌/密钥
4. **模型：** 点击获取模型列表/手动添加

::: warning 注意事项
- ✅ 确保 API 地址填写正确，包含 `/v1`
- ❌ 不要遗漏模型的前后缀
:::
