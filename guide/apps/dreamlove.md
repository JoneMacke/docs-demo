## 📝 配置步骤

1. **我的** → **API 配置** → **使用外部API互动**
2. **API 地址：** `https://ai.katioai.com`
3. **API Key：** 填入您的令牌/密钥
4. **模型：** 拉取模型列表

::: warning 注意事项
- ✅ 确保 API 地址填写正确，不包含 `/v1`
- ❌ 不要遗漏模型的前后缀
:::
